# The Home Kitchen Masterclass

A 12-week plant-based skills curriculum for Michael & Masa — built on ratios over
recipes, drilling weak spots, repetition, and inventing under constraint.

This is a single self-contained `index.html` (no build step, no dependencies)
served via GitHub Pages. It works as both a visual curriculum and a personal
workbook — progress, notebook, and invention log are saved locally in the
browser (`localStorage`), so it's a single-user tool per device.

## Develop locally

Open `index.html` directly, or serve it:

```
python3 -m http.server 8000
# then visit http://localhost:8000
```

## Source

`vegan-cooking-curriculum.md` holds the original written curriculum the site is built from.
